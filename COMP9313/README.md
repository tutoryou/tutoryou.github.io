# COMP9313

本课程介绍管理大数据的核心概念和技术。

主题包括：大数据和大数据分析的特征，存储系统（例如HDFS，S3），操纵大数据的技术（例如MapReduce，流式传输，压缩），编程语言（例如Spark，PigLatin），查询语言（例如Jaql， Hive），数据库系统（例如noSQL系统，HBase）和典型应用程序（例如推荐系统，降维，文本分析）。

作业信息：

作业安排包括四个projects，一个作业和最后的期末。作业是在一个虚拟机镜像linux环境下完成的。
